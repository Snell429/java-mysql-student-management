# 🎓 Student Management - Spring Boot

## 📌 Description
Application web développée avec Spring Boot permettant de gérer des étudiants, leurs classes et leurs cours à partir d'une base de données MySQL.

---

## 🛠 Technologies utilisées
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Thymeleaf

---

## 🗄 Base de données
- student
- class
- course
- enrollment

---

## 🔗 Fonctionnalités
- Affichage des étudiants
- Jointure entre plusieurs tables
- Affichage web avec Thymeleaf

---

## ▶ Lancer le projet

1. Configurer la base de données (MySQL)
2. Modifier `application.properties`
3. Lancer l'application

Accès :

http://localhost:7777/students
